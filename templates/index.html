About

This project intends to help developers in building the simplest hello world skeleton. The generated app will be in Flask and all this can be achieved with a single command.

Easy to install

How to install?

$ apt-get install python3-pip
$ pip3 install build-flask-app
$ build-flask-app --help

----------------------------------------------------

How to uninstall?

$ pip3 uninstall build-flask-app


Download the deb package here

How to install?

$ sudo dpkg -i build-flask-app_1.0_all.deb
$ build-flask-app --help

----------------------------------------------------

How to uninstall?

$ sudo apt-get remove build-flask-app

$ git clone https://github.com/askyourkode/build-flask-app
$ cd build-flask-app
$ python3 build-flask-app --help


Easy to use

root@70606kaf3efi:~

$ build-flask-app hello-world

$ cd hello-world/

$ python3 app.py
